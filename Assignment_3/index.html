What is the difference between id and class?

ID:ID it is used for an unique element and only one element should have the same id in a page. "#" symbol is used for ID
Class:Class it is used for a group of elements and can apply same class to many elements. "." symbol is used for Class

What is the CSS Box Model?

CSS box model is a box that wraps around every HTML element. It is used to create web designs and layout Every box consists of four parts: content, padding, borders and margins.

  • Content: The actual text, image, or data inside the element. Width and height apply to this area (by default).

  • Padding: Space between content and border.Inside the box. Increases the element size.

  • Border: Surrounds padding and content.You can change thickness, style, and color.

  • Margin: Space outside the border.Creates distance between elements.
